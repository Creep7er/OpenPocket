# OpenPocket 0.3.2

OpenPocket — экспериментальная виртуальная карманная консоль на Godot 4.7 с пиксельным интерфейсом, встроенными играми и приложениями, а также устанавливаемыми картриджами `.pctrg`.

Версия 0.3.2 является первым публичным снимком исходного кода. Проект находится в статусе Alpha.

## Возможности

- адаптивный Android-интерфейс и отдельный экран 400x320;
- управление с D-pad, A/B/X/Y, MENU и BACK;
- встроенные Snake, Pocket Pong и Pocket Notes;
- локальное хранение настроек и данных каждого пакета;
- локальный демонстрационный Store;
- импорт, проверка, обновление и удаление `.pctrg`;
- экспериментальные шаблоны SDK для игр и приложений.

## Установка картриджа

Откройте **Install Cartridge**, выберите `.pctrg`, проверьте сведения и подтвердите установку. Для вручную импортированного внешнего кода требуется Developer Mode.

Внешний картридж может выполнять GDScript внутри процесса OpenPocket. Sandbox и цифровых подписей пока нет; Developer Mode не делает код безопасным. Устанавливайте файлы только из доверенных источников. Подробнее: [SECURITY.md](../SECURITY.md).

## Запуск проекта

Установите Godot 4.7 stable без .NET и Python 3.10+:

```powershell
godot --path .
python tools/validate_project.py
godot --headless --path . --editor --quit
```
