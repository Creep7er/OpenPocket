# Installing

## Android

1. Download the `.pctrg` file.
2. Open PopugVPocket and choose `Install Cartridge`.
3. Select the file in Android Files, Downloads, Telegram, Drive, or another document provider.
4. Review identity, version, author, size, trust, checksum, and capabilities.
5. Enable Developer Mode for external code and confirm installation.

## Desktop

Choose `Install Cartridge`, select a `.pctrg` in the native file dialog, review it, and install. `user://imports` is only a developer fallback.
