# Quick Start

1. Clone `openpocket-app-template` or `openpocket-game-template`.
2. Change the reverse-DNS package id in `manifest.json`, `cartridge.json`, scene paths, and script constants.
3. Open `project.godot` in Godot 4.7 and run the sample.
4. Run `python tools/validate.py`.
5. Run `python tools/build.py`.
6. In OpenPocket choose `Install Cartridge`, select the output, review it, and install with Developer Mode enabled.

The normal first build takes 10-15 minutes including installing Godot and reading the sample.
