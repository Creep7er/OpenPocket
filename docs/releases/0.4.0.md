# OpenPocket 0.4.0

Status: feature branch, not yet released.

OpenPocket 0.4.0 introduces the static GitHub cartridge catalog, offline cache, local achievements, data-only themes/backgrounds, and permanent milestone rewards. Snake demonstrates the full flow with a length-40 Hard/Extreme completion challenge.

## Android

- `versionName`: `0.4.0`
- `versionCode`: `6`
- Package: `org.openpocket.app`
- New permission: `INTERNET` for catalog and release asset GET requests.

## Known Limitations

- External Godot code is not sandboxed or signed.
- Achievements are local and user-editable.
- Catalog availability depends on GitHub; the last successful cache remains browsable offline.
- Production signing is not configured.
