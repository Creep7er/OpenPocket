# PopugVPocket 0.5.0 - Reborn

Reborn introduces the PopugVPocket identity, VBoy and VGirl hardware profiles, V-Parrot, configurable direction controls, format v2 cartridges, and catalog moderation. It is an intentional compatibility reset from OpenPocket.

Android identity changes to `org.popugonet.popugvpocket`; old app data is not migrated automatically. See [Reborn Migration](../reborn-migration.md).

External cartridges remain unsandboxed. Production signing and broad real-device coverage are not yet complete.

The Reborn UI audit covers both physical profiles plus Shell, system overlay, Snake, Pong, Notes, and Breakout menus. VGirl uses left directions, a centered display, right action buttons, and centered MENU/BACK controls. Breakout modal selection rows are clipped to their dialog instead of spanning the virtual screen. Android adaptive icon artwork uses a padded V-Parrot foreground that remains readable under common launcher masks.
