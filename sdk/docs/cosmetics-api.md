# Cosmetics

SDK 0.5.0 manifests may declare data-only `theme` and `background` cosmetics. Definitions and assets must use safe relative paths under the cartridge root. Cosmetic scripts, shaders, layout changes, permissions, and platform settings are not accepted.

Provided cosmetics disappear with an uninstalled cartridge. Permanent rewards are issued only by a declared achievement and copied into the Reward Vault, where they survive cartridge updates, uninstall, and package save removal.
